## 三、代码审查发现的 Bug

### 🔴 P0 — 运行时 Bug

**`js/juice.js` L291: `updateJuiceOnly` 缺少右花括号**

```javascript
// L289 — beams 循环缺少 } 闭合
for (let i = j.beams.length - 1; i >= 0; i--) {
  j.beams[i].life -= dt;
  if (j.beams[i].life <= 0) j.beams.splice(i, 1);
for (let i = j.smokePuffs.length - 1; i >= 0; i--) {   // ← 嵌套在 beams 循环内！
```

**影响**：烟雾粒子循环被错误嵌套在光束循环内部。若 `j.beams` 为空数组，烟雾粒子永远不会更新。光束数量>0时烟雾粒子每帧被更新 N 次（浪费计算）。

**修复**: 在 L291 `if (...) j.beams.splice(i, 1);` 后加 `}`。
