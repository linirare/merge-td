# 武器重设计 — v61 chibi 原创设计

## 设计原则

1. **一眼认出**：靠轮廓/剪影区分，不靠细节
2. **chibi 比例**：武器相对于头(headR)来定尺寸，大头配大武器，小身体衬托
3. **粗线条**：Canvas 2D 纯几何绘制，线宽要够粗才看得见
4. **L1 headR≈10px 基准**：武器关键尺寸 15-35px 区间

## 5 种武器设计

### 1. 盾 — Tank（西瓜/椰子/草莓/牛油果）

**概念**：齿轮太阳盾
- 圆盾带 8 个三角齿外圈，像齿轮/太阳光芒，剪影独一无二
- 半径 = headR × 1.2（L1≈12px）
- 中心一个大铆钉（实心圆）
- 齿尖到圆心画细辐条线
- 攻击时整个盾前顶 + 齿尖发白光

**区别于传统圆盾**：齿轮边缘比普通圆盾识别度高得多

### 2. 枪 — Front（菠萝/火龙果）

**概念**：三叉戟
- 杆长 headR × 2.0，中间主刃 + 两侧小倒钩
- 三叉戟的剪影比单头枪更有辨识度
- 杆尾有一个小球（平衡锤）
- 攻击时三叉前刺 + 中间刃发亮
- 倒钩方向朝外，形成 W 形尖端

**区别于普通枪**：三叉让人想起海神戟，剪影突出

### 3. 剑 — Rush（香蕉/柠檬/橄榄）

**概念**：柴刀/弯刀
- 宽刃厚背，刃尖上翘（反曲刀造型）
- 刃长 headR × 1.6，刃宽 headR × 0.5
- 刀背是粗厚的深色边，刀刃是亮色金属
- 缠绳握柄（交叉线纹理）
- 攻击时横扫 + 刀尖拖出弧形白线

**区别于普通剑**：宽短弯刀 vs 细长直剑，轮廓完全不同

### 4. 弓 — Back（葡萄/蓝莓/芒果/樱桃/冰梨/哈密瓜/蜜桃）

**概念**：不对称长弓
- 上臂比下臂长（日式弓造型），弓身总高 headR × 3.0
- 上臂占 60%，下臂占 40%
- 弓臂外侧有节（竹节凸起），不是光滑弧线
- 双股弦（两条平行细线）
- 拉弓时箭杆可见，带十字尾羽
- 发射后弓臂回弹振动（快速来回微旋）

**区别于普通弓**：不对称 + 竹节 + 双弦，不像量产武器

### 5. 炮 — Siege（橙子/南瓜）

**概念**：短口迫击炮
- 炮管短粗（长 headR × 1.2），炮口喇叭形外扩
- 炮口直径是炮管 2 倍，像漏斗
- 底座是一个三角形支架（两腿撑地）
- 炮身上有箍环（2-3 条横线）
- 攻击时炮口喷出锥形火焰 + 整炮后坐
- 后坐后缓慢复位

**区别于普通炮**：喇叭口 + 三角架 + 短粗身 = 迫击炮剪影，不是长管加农炮

## 尺寸汇总（L1 headR≈10）

| 武器 | 关键尺寸 | L1 值 | 视觉特征 |
|------|---------|-------|---------|
| 盾 | 半径 headR×1.2 | 12px | 8齿齿轮边+中心铆钉 |
| 枪 | 杆长 headR×2.0 | 20px | 三叉W形尖端+杆尾球 |
| 剑 | 刃长 headR×1.6 | 16px | 宽背弯刀+缠绳柄 |
| 弓 | 总高 headR×3.0 | 30px | 不对称上长下短+竹节+双弦 |
| 炮 | 管长 headR×1.2 | 12px | 喇叭口+三角架+箍环 |

## 与身体的比例关系

L1 身体总高(脚到肩) ≈ s×8.6 ≈ 16px
弓(30px) > 身体 > 剑(16px) ≈ 炮(12px) ≈ 盾(12px)
弓是身上最显眼的装备，一眼认出后排；盾剑炮跟身体同大小，协调；枪 20px 比身体略长，显攻击距离。
