# 第八轮：启动验证 + 最终完整性检查

## 验证结论：服务器当前无法启动

### 致命阻塞：JWT_SECRET 未设置

`server/auth.js:4-6` 要求 `JWT_SECRET` 环境变量必须存在，否则直接 `throw`。项目没有 `.env` 文件、没有 `dotenv` 包、Railway 部署脚本也不传这个变量。**服务器起不来。**

### 次要阻塞：无优雅关闭

没有 `SIGTERM`/`SIGINT` 处理。Railway 重新部署时 PvP 房间状态直接丢失、进行中的 HTTP 请求被截断。
